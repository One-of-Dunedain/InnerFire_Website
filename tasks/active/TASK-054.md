## Context

We have 3 demo videos on the homepage carousel and 1 in the blog article. Current problems:
- Videos are 14-15MB each (44MB total) — terrible mobile performance, loop stutter
- No preview/fullscreen separation — same heavy file for both
- No visual hint that videos are tappable
- Blog video container `.media-video-v-shell` looks inconsistent with site style

## Two-tier architecture

Each video has TWO files:
- `demo-X-preview.mp4` — low-res (480x854), compressed, ~500KB-1MB. Used for muted autoplay loop in cards.
- `demo-X.mp4` — original quality (14-15MB). Loaded on-demand when user taps to fullscreen.

The preview file is set as `<video src>`. The original file path is stored in `data-fullsrc` attribute. On tap, JS swaps the source and opens fullscreen.

## 4. JavaScript changes — script.js

Replace current fullscreen logic with two-tier swap:

```js
function openVideoFullscreen(video, card) {
  if (!video) return;

  var fullSrc = video.getAttribute('data-fullsrc');

  // Swap to full quality source if available
  if (fullSrc && video.currentSrc !== fullSrc) {
    var currentTime = video.currentTime;
    video.src = fullSrc;
    video.currentTime = currentTime;
  }

  video.muted = false;
  video.defaultMuted = false;
  video.volume = 1;
  video.controls = true;

  if (video.requestFullscreen) {
    var fsPromise = video.requestFullscreen();
    if (fsPromise && typeof fsPromise.catch === 'function') {
      fsPromise.catch(function() {});
    }
  } else if (video.webkitRequestFullscreen) {
    video.webkitRequestFullscreen();
  } else if (video.webkitEnterFullscreen) {
    video.webkitEnterFullscreen();
  }

  var playPromise = video.play();
  if (playPromise && typeof playPromise.catch === 'function') {
    playPromise.catch(function() {});
  }
}

function restoreInlineState(video) {
  if (!video) return;

  var fullSrc = video.getAttribute('data-fullsrc');
  var previewSrc = video.querySelector('source');

  // Swap back to preview source
  if (fullSrc && previewSrc && video.currentSrc === fullSrc) {
    video.src = previewSrc.src;
  }

  video.controls = false;
  video.muted = true;
  video.defaultMuted = true;
}
```

Also add the same tap-to-fullscreen logic for `.video-card` elements (blog):
```js
document.querySelectorAll('.video-card').forEach(function(card) {
  var video = card.querySelector('.video-card-player');

```
